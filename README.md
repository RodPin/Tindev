# Tindev
Omnistack 8.0 

# how to use
### Run the server
at /Tindev/backend
 ```
   cd backend && yarn && yarn dev 
 ```
### Run web front end
at /Tindev/webfrontend
 ```
   cd webfrontend && yarn && yarn start
 ```
### Run Mobile App
  at /Tindev/tidevRN
   ```
   cd tindevRN && yarn && react-native run-android
    
```
 
Make emulator recognize port 3333
at /Tindev/tidevRN
```
    adb reverse tcp:3333 tcp:3333
    
```
